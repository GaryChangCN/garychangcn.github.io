ES6-Map

ES6引入Map的目的

js 的对象本质上就是键值对集合也就是 Hash结构，但是只能用字符串作为键值，而 Map 就可以解决这个问题。

用法

Map 可以接受一个数组作为参数，该数组成员是一个个表示键值对的数组。

实例属性及操作方法

size 属性

返回 Map 结构成员总数。

遍历方法

  • keys() 返回键名的遍历器。
  • values() 返回键值的遍历器。
  • entries() 返回所有成员的遍历器。
  • forEach() 遍历 Map 的所有成员。
