ES6 promise 对象

含义

  • 状态

    promise 对象代表一个异步操作，有三种状态：

    • Pending 进行中
    • Resolved 已完成 （又称Fulfilled）
    • Rejected 失败

  • 特点
    一旦状态改变就不会在改变。

目的

解决回调写法的混乱以及一个统一标准。
